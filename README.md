
# Portal Estiba – PWA PRO

Incluye:
- Gestión de jornales por quincenas
- Añadir / editar / borrar jornales
- Campos: fecha, precio, especialidad, barco, empresa, nº parte
- Selector de mes y año
- Resumen mensual automático
- Cálculo bruto y neto por IRPF
- Exportar a CSV (Excel) y PDF
- UI tipo app (tarjetas, colores, iconos)
- Guardado en LocalStorage
- PWA instalable

Proyecto 100% local, sin datos reales.
